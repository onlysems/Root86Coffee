📤 Export & Save Changes

After making changes, click Generate Export to get the updated coffees.js file content, then follow the steps below to publish your changes to GitHub.

How to publish your changes:
  1. Click Generate Export then Copy to Clipboard
  2. Go to your GitHub repository page
  3. Open the file js/coffees.js
  4. Click the pencil ✏️ icon to edit it
  5. Select all the existing content (Ctrl+A) and paste your copied content
  6. Scroll down and click Commit changes
  7. Your website will update within 1–2 minutes!
